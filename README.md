# HR-Attrition-Analysis

1. **Title: HR Attrition Analysis Dashboard**
This project presents a comprehensive HR Analytics Dashboard focused on understanding employee attrition trends within an organization. The dashboard enables HR teams and key decision-makers to identify major factors contributing to employee turnover and uncover actionable workforce insights.

2. **Project Overview:**
Employee attrition is a critical challenge for many organizations, impacting productivity, morale, and revenue.
This dashboard consolidates and visualizes key HR metrics to answer questions such as:

i.  How many employees are leaving?
ii. Which departments experience the highest attrition?
iii.Which job roles, age groups, or salary ranges are most affected?
iv. Are there patterns around gender, education, or tenure?
v.  By identifying high-risk segments, organizations can proactively improve retention strategies.
